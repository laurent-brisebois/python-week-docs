Python

Python is an interpreted, high-level, general-purpose programming language. It is one of the most popular programming languages in the world, and is easy to learn.

Git

Git is a distributed version-control system for tracking changes in source code during software development. It is designed for coordinating work among programmers.

Bash

Bash is a command processor that typically runs in a text window where the user types commands that cause actions.
